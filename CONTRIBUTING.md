# Contributing to ZENTRA

Thank you for your interest in contributing to ZENTRA.

ZENTRA is a behavioral financial infrastructure project focused on
risk intelligence, behavioral financial signals, and systemic
financial observation.

---

## Contribution Scope

At this stage, ZENTRA accepts limited contributions related to:

• documentation improvements  
• developer tooling  
• visualization components  
• demo interface improvements  

Core financial models and infrastructure components are private.

---

## Contribution Process

1. Fork the repository
2. Create a new branch
3. Submit a pull request describing the improvement

## Code Guidelines

Contributions should follow these principles:

• clean and readable code  
• minimal external dependencies  
• security awareness  
• no exposure of sensitive financial logic  

---

## Security Disclosure

If you discover a security issue, do **not** open a public issue.

Please report it directly:

contact@zentrarisk.com

---

## Project Direction

ZENTRA focuses on building a behavioral financial intelligence layer
that operates across multiple financial institutions.

The platform includes:

• RiskLens behavioral scoring  
• Credit Stress modeling  
• Lens marketplace intelligence  
• Financial telescope macro observation
