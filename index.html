RiskLens API

AI-powered modular risk scoring infrastructure. API-first engine for banks, neobanks, and embedded finance platforms.

Open RiskLens OpenAPI JSON Health

curl -X POST "https://zentrarisk.com/v1/risk" -H "Content-Type: application/json" -H "X-API-Key: zentra-demo-key" -d '{"amount":1000,"sector":"retail","sector_risk_level":2,"payment_delay_days":10,"customer_score":78,"exposure_ratio":0.25}'

Note: If RiskLens API is not hosted on the root domain, these buttons must point to the RiskLens Render URL (or risk.zentrarisk.com once DNS is added).

Credit Stress Engine
