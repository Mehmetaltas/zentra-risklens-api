Aktif Ürünler (Live)

RiskLens API

AI destekli modüler risk skorlama altyapısı. Bankalar / neobankalar / embedded finance için API-first risk motoru.

Open RiskLens OpenAPI JSON Health
curl -X POST "https://zentrarisk.com/v1/risk" \ -H "Content-Type: application/json" \ -H "X-API-Key: zentra-demo-key" \ -d '{"amount":1000,"sector":"retail","sector_risk_level":2,"payment_delay_days":10,"customer_score":78,"exposure_ratio":0.25}'

Not: “risk.zentrarisk.com” DNS hazır olduğunda bu butonlar oraya yönlenecek.
